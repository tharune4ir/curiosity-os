## 🏗️ Core Component Strategy

### 1. The 3D Neural Engine (`UniverseGraph.tsx`)
- **V1 Role:** The "Sun" around which the whole app orbited.
- **V2 Evolution:** Moves to the background. It becomes a **"Visual Context Overlay"** for teachers—useful for seeing "where this activity lives in the big picture," but no longer the main interaction hub.

### 2. The Neural Core (`NeuralCore3D.tsx`)
- **V1 Role:** Atmospheric visual hero on the homepage.
- **V2 Evolution:** Preserved as a high-end visual reward or "Loading/Standby" visual state, but de-emphasized to prioritize data clarity.

### 3. The Data Structure (`CURIOSITY_SCHEMA_V1.md`)
- **V1 Role:** Mapped 147 Nodes and Edges.
- **V2 Evolution:** **Stays as the hidden "Logic Layer."** Every V2 Activity will "hook" into these nodes to pull descriptions and prerequisites, keeping the Intelligence Engine robust behind-the-scenes.
