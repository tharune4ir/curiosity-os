# Signal vs Noise: The Two-Class Mystery

## The Run Mode
### 1. The Setup
Hand out scorecard data for Class A and Class B. 
- Class A: 50, 50, 50, 50, 50 (Average 50)
- Class B: 0, 0, 50, 100, 100 (Average 50)

### 2. The Mission
Students must decide: "Which class is better for a teamwork-based project?" and "Which class is better for a top-5 competition?" 
- Justify the choice without using the word 'average'.
- Sketch the distribution on a simple number line.

### 3. The Decompile
Focus on the **Hidden Structure**:
- A single number (the average) hides the variance.
- Reality is often about the **distribution**, not the **summary statistic**.
