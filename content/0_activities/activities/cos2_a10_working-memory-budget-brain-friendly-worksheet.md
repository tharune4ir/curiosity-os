# Working Memory Budget: Build a Brain-Friendly Worksheet

## The Run Mode
### 1. The Setup
Hand out a 'messy' worksheet or textbook page.
- **The Load Map:** Students mark red (hard/noisy) and green (easy/clear).

### 2. The Mission
Pairs have 25 minutes to rewrite/redesign the material:
1. **Reduce Split Attention:** Move labels next to the parts they describe. 
2. **Chunking:** Group the 10 steps into 3 phases.
3. **Externalize:** Add a small 'scratch box' where intermediate steps can be written.

### 3. The Decompile
Focus on **Cognitive Load**:
- Hard learning is good; hard *reading* is unnecessary friction.
- Your brain's 'bucket' (working memory) is small—protect it.
