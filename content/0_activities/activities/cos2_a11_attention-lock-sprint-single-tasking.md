# Attention Lock Sprint: Single-Tasking Under Noise

## The Run Mode
### 1. The Setup
Run two rounds using the same short list of tasks:
- **Round 1 (Friction):** Background chatter, frequent interruptions by teacher.
- **Round 2 (Lock):** Total silence, single task, visible timer.

### 2. The Mission
In each round, students have 5 minutes to complete as many tasks as possible.
1. Compare accuracy, speed, and felt strain.
2. Sign an 'Attention Contract' for one week.

### 3. The Decompile
Focus on **Attention Control**:
- Switching attention has a 'cost' that slows down the whole brain.
- Single-tasking is the real superpower in a noisy world.
