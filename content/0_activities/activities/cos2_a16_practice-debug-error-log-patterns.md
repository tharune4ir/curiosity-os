# Practice Debug: Error Log Patterns (Beyond 'Silly Mistakes')

## The Run Mode
### 1. The Setup
Bring a recent test or homework with errors.
- **The Catch:** No 'X' marks. Just circles.

### 2. The Mission
Students follow the **Practice Debug Protocol**:
1. **Name the Error:** Give it a funny but precise name (e.g., 'The Sign Swapper,' 'The Unit Ghost,' 'The Read Slider').
2. **Identify the Trigger:** What happened right before the error? (e.g., 'I was rushing,' 'I saw a similar word').
3. **The Pre-Flight Check:** Write a 1-sentence reminder for next time (e.g., 'Check units BEFORE starting calc').

### 3. The Decompile
Focus on **Pattern Recognition**:
- You don't make 100 random mistakes; you make 3-4 patterns repeatedly.
- Debugging is how experts stay accurate under pressure.
