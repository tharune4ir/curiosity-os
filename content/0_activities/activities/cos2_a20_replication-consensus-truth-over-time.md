# Replication & Consensus: Truth Over Time

## The Run Mode
### 1. The Setup
Each of 5 groups is given the same "Phenomenon" (e.g., "What is the average height of students in this school?"). 
- **The Catch:** Each group only gets a small, biased sample (e.g., only the basketball team, only class 7, only class 10).

### 2. The Mission
Groups work in rounds:
1. **Round 1:** Each group publishes its "Result" independently. (Results will be wildly different).
2. **Round 2:** Groups see Group 1's result. Do they update their own? 
3. **Round 3:** All 5 groups publish results on a single **Consensus Board**.
4. **Conclusion:** What is the "Stable Truth"?

### 3. The Decompile
Focus on **Scientific Consensus**:
- One study is a signal; 5 studies are a consensus.
- Outliers are normal in small samples.
