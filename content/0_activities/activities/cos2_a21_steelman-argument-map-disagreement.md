# Steelman & Argument Map: Disagreement Studio

## The Run Mode
### 1. The Setup
Each pair picks a topic with two strong sides. 
- **The Catch:** No debating until the Steelman phase is complete.

### 2. The Mission
Pairs follow the **Steelman Protocol**:
1. **The Argument:** Write one side's claim. 
2. **The Steelman:** Students rebuild the opposing claim into its most logical, evidence-based version. (e.g., "If I were them, I would use this data to say...").
3. **The Argument Map:** Draw a tree showing the **Claim → Sub-claim → Evidence** for both sides.
4. **Conclusion:** Find the one specific place where they disagree on **Values** vs. **Facts**.

### 3. The Decompile
Focus on **Cognitive Empathy**:
- Understanding the other side doesn't mean agreeing with them.
- Truth-seekers map the whole territory; debaters only map their own camp.
