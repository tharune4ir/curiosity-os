## The Invisible Rule
Meta-analytic research on concept/knowledge maps shows they can improve learning and help students build structure from information

## The Matrix Trap
You only read line-by-line. You never see how ideas connect, so you forget or confuse topics.

## The Escape Hatch
Draw a concept map: central idea → branches → sub-branches → examples. Add arrows like “causes,” “depends on,” “leads to.”

## The Superpower
You revise faster because you can ‘zoom out’ and see the chapter like a map.

## The Level Up Action
Make a mini concept map for one topic using only 7 boxes and 6 arrows.
