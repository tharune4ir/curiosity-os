## The Invisible Rule
Dual coding theory says we can store info as words and as images. Using both can improve recall because you get two routes to the same idea

## The Matrix Trap
You keep only text notes. Later, everything looks the same and nothing stands out.

## The Escape Hatch
Add tiny visuals: arrows, boxes, a quick sketch, a simple diagram. Not art—just a memory handle.

## The Superpower
You recall faster in exams because you can ‘see’ the answer structure in your head.

## The Level Up Action
For one topic today, draw one 30-second diagram (even ugly).
