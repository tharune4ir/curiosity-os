## The Invisible Rule
Keyword mnemonics can help for some tasks (like vocabulary), but research reviews warn it doesn’t generalize perfectly to everything and can waste time if overused

## The Matrix Trap
You make mnemonics for every single line and spend more time creating jokes than learning.

## The Escape Hatch
Use mnemonics only for ‘stubborn’ items: formulas, dates, scientific names. Then test with active recall the next day

## The Superpower
You save time and still get the memory boost where it matters.

## The Level Up Action
Pick ONE tough term today and create ONE mnemonic. Then test it from memory after 10 minutes.
