## The Core Joy
Family games create belonging + laughter. When competition stays playful, it becomes a memory machine.

## The Memory Maker
Cousins visit in holidays. You do silly events: spoon race, pillow dodge, cricket one-hand catch challenge. Everyone screams like it's IPL finals.

## The Hidden Upgrade
You learn leadership, fairness, and how to include younger kids and older cousins without making anyone feel small.

## The Matrix Trap
Cousins visiting but everyone only watching separate videos, then saying 'time flew' with no real memory.

## The Weekend Mission
Create 3 events that need zero equipment. Make simple rules + one funny prize (like 'Snack King'). Take one group photo.
