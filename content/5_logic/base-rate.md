## The Cheat Code
Base rate is the overall frequency of something. Always check the general odds first. For example, if 90% of students pass an exam, start with the idea that a random student likely passes.

## The Casino Trap
Ignoring base rates can fool you. If one student got 100/100, you might think everyone will ace it. But if only 10% normally score full, odds say most won’t.

## The Real World Hack
Your school has only 5% seats on the sports team. Even if you are good, remember most tryouts fail. Don’t assume you’ll make it just because you like the sport.

## The Superpower
You judge chances correctly by remembering the averages. You won’t be misled by rare events.

## The Mission
Find a statistic about something you care about (like pass rates) and use it next time to set real expectations.
