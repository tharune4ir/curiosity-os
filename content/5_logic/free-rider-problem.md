## The Cheat Code
Free rider problem is when someone benefits from group work without contributing. It hurts groups because some do all work while others enjoy results for free.

## The Casino Trap
You might think 'I’ll let others do it'. But if everyone thinks that, projects fail because nobody helps.

## The Real World Hack
In a group study, one friend doesn’t prepare but still passes by copying others. Everyone works hard, and one person gets the benefit without effort.

## The Superpower
You make sure you contribute, and you know when to address freeloaders. You keep group work fair.

## The Mission
Next group project, do your fair share. Notice if others do too.
