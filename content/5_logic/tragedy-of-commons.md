## The Cheat Code
Tragedy of the Commons is when individuals use a shared resource selfishly, causing it to deplete and harming everyone.

## The Casino Trap
You might think 'I can use more because I’m just one person'. But if everyone thinks that, the resource runs out.

## The Real World Hack
Sharing a cricket pitch: if one person hogs time and practice, others get less. If everyone hogs, the pitch gets ruined.

## The Superpower
You keep community resources safe. You either share fairly or set rules so everyone benefits.

## The Mission
Next time you use a shared resource (like a computer or field), use only your fair share and see if it helps others too.
