## The Invisible Architecture
Open source software is code that is publicly shared — anyone can read it, use it, and contribute improvements. Linux (the operating system running most of the world's servers), Python, React, Android — all open source. Tens of thousands of developers around the world, most unpaid, collectively maintain these projects on GitHub. When a contributor finds a bug, they 'fork' the code (make their own copy), fix it, and submit a 'pull request' — asking the maintainers to accept their fix into the main project.

## The Consumer Trap
Every app on your phone uses open source code. WhatsApp's servers run Linux. Instagram's app uses open source React Native libraries. Zomato uses open source databases. These trillion-rupee companies are built on top of code that thousands of anonymous developers freely shared. Almost nobody knows this.

## The Builder Hack
A builder submits their first pull request to any open source project — even fixing a typo in documentation. This appears on their GitHub profile forever. For internship and job applications, a public GitHub profile with real contributions is worth more than any certificate.

## The Superpower
You participate in the global system that builds shared digital infrastructure. You go from being a consumer of open source to a producer. Your code can run on millions of computers worldwide, and your name is in the git history.

## The First Line Of Code
Go to github.com/explore. Find a project with 'good first issue' tags. Read the issue. Even if you can't solve it, reading a real-world problem solved by open source contributors is one of the best programming educations available.
