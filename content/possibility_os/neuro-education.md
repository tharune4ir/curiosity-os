## The Matrix Truth
Using neuroscience to teach better: memory techniques, brain-training games, or personalized learning based on how people think.

## The Leverage
If you apply brain science to learning, you can dramatically improve education. Corporations and schools pay for evidence-based methods.

## Skill Stack
Neuroscience, pedagogy, data analysis, software development.

## Economic Reality
Educational psychologist, edtech founder, curriculum designer. Mixed funding (schools, grants, startups).

## Proof of Work
Test a learning technique on yourself: for example, try spaced repetition for vocabulary and track retention improvement.
