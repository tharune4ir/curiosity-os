## The Matrix Truth
Cities will be run on data: smart traffic lights, connected public transport, waste sensors. Urban planning will use real-time data for efficiency.

## The Leverage
If you can build tech for cities, you improve millions of lives and can make huge projects. City contracts can fund large ventures.

## Skill Stack
IoT, data analytics, urban planning, public policy.

## Economic Reality
Smart city consultant, municipal CTO, urban data scientist. Collaboration with governments, mixed public-private funding.

## Proof of Work
Design a plan: pick a common city problem (traffic, waste) and outline a connected solution using sensors and data analysis. Mock up the data flow.
