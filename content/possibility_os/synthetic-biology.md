## The Matrix Truth
Biology is becoming programmable. We can treat DNA like code to create organisms that clean up waste or produce new materials.

## The Leverage
Writing life code unlocks huge power. Anyone who masters this writes the software of nature, solving diseases and food shortages.

## Skill Stack
Molecular biology, genetic editing (CRISPR), bioinformatics, Python.

## Economic Reality
Bioengineer, biotech entrepreneur, researcher. Big R&D budgets, high risk and reward.

## Proof of Work
Fetch a public genome file and write a script that identifies a useful gene pattern. Publish your code.
